# Goods4you

## Stack:

- React
- TypeScript
- CSS modules
- React-router-dom

## Для начала работы необходимо:

1. Сохранить проект локально (save the project locally):

```bash
git clone https://github.com/yazakhariya/TASKTI.git
```

2. Ввести в консоли команду (type the command):

```bash
npm run dev
```
